## Key Implementation Details

### API Data Flow

The component calls `DashboardApiService.getSummary()` on initialization and stores the result in Angular signals for loading, error, and summary state. Retry uses the same load path, keeping the behavior small and predictable.

### Display Consistency

The dashboard reuses existing formatting and label helpers from friction and initiative features. This keeps priority/status/category labels and tag severities aligned across screens.

### MVP UI Scope

The dashboard uses compact operational cards and a table rather than adding charts or decorative visuals. This matches the MVP goal of helping teams quickly scan impact and priority.
